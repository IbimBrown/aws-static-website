AWS S3 Static Website
This project demonstrates how to host a static website using Amazon S3.

Project Overview
The website is built using basic HTML and CSS and hosted on an S3 bucket configured for static website hosting.

AWS Services Used
Amazon S3 (Static Website Hosting)

Configuration Steps
Created an S3 bucket
Uploaded static website files (HTML/CSS)
Enabled static website hosting
Set index document to `index.html`
Applied a public bucket policy to allow read access

Live Website
http://my-first-aws-static-site-2705.s3-website.eu-north-1.amazonaws.com

What I Learned
- How Amazon S3 stores and serves static content
- Bucket policies and public access settings
- Basics of cloud-based website hosting
